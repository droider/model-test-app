# Model Test App

A React web interface for testing different AI models via Scaleway's OpenAI-compatible API.

## Getting Started

1. Clone the project or unzip the provided folder.
2. Run:

```bash
npm install
npm run dev
```

3. Update your `App.jsx` to insert your actual IAM API key from Scaleway.

## GitHub Pages Deployment

GitHub Actions will automatically deploy the app on every push to `main`.

Live URL:
https://droider.github.io/model-test-app/
